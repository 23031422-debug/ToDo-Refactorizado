# ToDo-Refactorizado

Aplicación de lista de tareas desarrollada para el **Lab U3-01** de Ingeniería de Software.

## Funcionalidades

| Funcionalidad | 
| Agregar tareas (título + descripción) | 
| Marcar como completadas |
| Eliminar tareas | 
| Persistencia con LocalStorage | 
| Filtros (All/Completed/Uncompleted) | 
| Búsqueda por palabras | 
| Pruebas unitarias (TDD) | 

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API

## Conceptos aplicados

- ✅ localStorage (getItem, setItem, JSON)
- ✅ Eventos del DOM (addEventListener)
- ✅ Manipulación del DOM (creación, inserción, remoción)
- ✅ Estructura del programa (separación de responsabilidades)
- ✅ Validaciones (campo vacío)
- ✅ TDD (pruebas antes del código)

##  Ejecutar pruebas

Las pruebas se ejecutan automáticamente al cargar la página.
Abre la consola (F12) para ver los resultados detallados.
